A 65% made by NEO Studio.

* Keyboard Maintainer: [QwertyKeys](https://github.com/qwertykeys-official)
* Hardware Supported: neo PCB
* Hardware Availability: IC: https://qwertykeys.notion.site/Neo65-Core-Plus-23a3d0900942809dbd5cfadf9cc401f1

Make example for this keyboard (after setting up your build environment):

    make neo/neo65p_wired:default

Flashing example for this keyboard:

    make neo/neo65p_wired:default:flash

**Bootloader:** Press the physical reset button on the bottom side of the PCB (labeled SW1) or hold down ESC while plugging in the board.
